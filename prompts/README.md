# Prompts

Deze map is gereserveerd voor Codex-, ChatGPT- en VS Code-agentprompts.

## Richting

Voeg prompts alleen toe via een expliciete prompt-engineering sprint. Elke prompt moet aangeven:

- agentrol
- taaktype
- toegestane bestanden
- verboden bestanden
- validatieregels
- eindrapportformaat
